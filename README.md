# Vue2.5开发去哪儿网App
**技术栈和主要框架**

📦 Vue 全家桶：vue2.5 + vuex + vue-router + webpack

📌 ES6

📡 网络请求：axios

 **项目运行**
```bash
# 克隆到本地
git clone https://github.com/1417766861/Vue2.5-App.git

# 安装依赖
npm install

# 开启本地服务器localhost:8088
npm run dev

# 发布环境
npm run build
```
